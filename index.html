Proyectos

En esta sección, encontrarás los proyectos más destacados de mi portafolio.

Google Clone

Este fue mi primer proyecto que creé al iniciar como frontend developer. Esta es una réplica que hice de la página de inicio de Google Chrome. Es un sitio web estático hecho con HTML y CSS, lo diseñé con el objetivo de poner en práctica lo que había aprendido.

Eco Store Sass

Eco Store es una página web que simula una tienda en línea que ofrece una amplia variedad de productos ecológicos para ayudarte a vivir de manera más sostenible y respetuosa con el medio ambiente. Este sitio web fue construido con Sass, CSS y HTML.

JSTODOAPP

JSTODOAPP es una página web de gestión de tareas desarrollada en JavaScript. Permite crear una lista de tareas para ayudar a los usuarios a organizar sus actividades. Es una página web de código abierto disponible para su uso en cualquier dispositivo.

Platzi Travel

Este proyecto es una representación de un sitio web de reserva de viajes donde los usuarios pueden explorar destinos y planificar sus próximas vacaciones. Este es el primer proyecto que incluye un modo oscuro. Está construido con el framework Tailwind CSS y HTML. Se utilizaron propiedades de CSS como display grid y flex para la organización de las tarjetas.

Tailwind Clone Homepage Mario 3D Land

Esta es una página de inicio que recrea la página principal del videojuego Super Mario 3D Land. Está hecha con HTML y para la creación de los estilos se utilizó el framework Tailwind CSS.

Smash Pig

Smash Pig es un minijuego en el cual tienes que hacer clic en cada cerdo para poder obtener puntos. Este proyecto está hecho con el preprocesador Sass y tiene animaciones. Lo desarrollé con la intención de poner en práctica la creación de animaciones.
