Proyectos

En esta sección, encontraras los proyectos mas destacados de mi Portafolio

Google-clone

Este fue mi primer proyecto que cree al iniciar como frontend developer. Esta es una replica que cree de la pagina de inicio de google chrome es un sitio web estático. Está hecho con HTML y CSS, lo diseñé con el objetivo de poner en práctica lo que había aprendido.

Eco Store Sass

Eco Store es una pagina web que simula una tienda en línea que ofrece una amplia variedad de productos ecológicos para ayudarte a vivir de manera más sostenible y respetuosa con el medio ambiente. Este sitio web fue construido con Sass, css y html

JSTODOAPP

JSTODOAPP es una pagina web de gestión de tareas desarrollada en JavaScript, permite crear una lista de tareas para ayudar a los usuarios a organizar sus tareas. Es una pagina web de código abierto que está disponible para su uso en cualquier dispositivo.

Platzi Travel

Este proyecto es una representación de un sitio web de reserva de viajes donde los usuarios pueden explorar destinos, y planificar sus próximas vacaciones. Este es el primer proyecto que tiene modo oscuro. Esta construido con el framework Tailwind css y html. Se uso las propiedades de css, display grid y flex, para la organización de las cards.

Tailwind clone homepage Mario 3d land

Esta es una homepage que recrea la pagina principal del videojuego Super Mario 3D land. Esta hecha con html y para la creación de los estilos se uso el framework tailwind css.

Smash Pig es un minijuego en el cual tienes que hacer click en cada cerdo para poder obtener puntos. Este proyecto esta hecho con el preprocessor Sass, y tiene animaciones. Este proyecto lo desarrollé con la intención de poner en practica la creación de animaciones.
