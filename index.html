Proyectos

Google Clone Home Page

Este fue mi primer proyecto que cree al iniciar como frontend developer. Esta es una replica que cree de la pagina de inicio de google chrome es un sitio web estatico. Está hecho con HTML y CSS, lo diseñé con el objetivo de poner en práctica lo que había aprendido.

Eco store Sass

Eco Store es una pagina web que simula una tienda en línea que ofrece una amplia variedad de productos ecológicos para ayudarte a vivir de manera más sostenible y respetuosa con el medio ambiente. Este sitio web fue construido con Sass, css y html

Platzi Travel

Bienvenido a Platzi Travel, la agencia de viajes ficticia que te llevará a aventuras emocionantes por todo el mundo. Este proyecto es una representación de un sitio web de reserva de viajes donde los usuarios pueden explorar destinos, y planificar sus próximas vacaciones. Este es el primer proyecto que tiene modo oscuro. Esta construido con el framework Talwind css y html.
